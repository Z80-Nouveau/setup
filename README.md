# Z80-Nouveau Project Setup

The software and hardware for this project can be developed on a Raspberry PI or similar Unix/Linux system.

*Note that this project only uses a PI to program the FPGA.  But that can be done with a JTAG cable if you are familiar with doing so.*

- [raspberry-pi-install](https://github.com/johnwinans/raspberry-pi-install) - How to prepare a Raspberry PI to host everything.
- [nouveau-install](./nouveau-install.md) - How to install all the parts on the Nouveau project onto a PI or Linux box.
